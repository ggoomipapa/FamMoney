============================================
현재 문제
============================================
Android Studio에서 Gradle 캐시 파일 잠금 오류 발생
→ 컴퓨터 재부팅으로 해결 예정

============================================
재부팅 후 해야 할 일
============================================

### 1. Android Studio에서 빌드 확인
- 프로젝트 열기
- Build > Rebuild Project 실행
- 정상 빌드 확인

### 2. SHA-1 인증서 확인
- Android Studio > View > Tool Windows > Gradle
- SelectMoney > app > Tasks > android > signingReport 더블클릭
- 출력창에서 SHA1 값 복사

### 3. Firebase 프로젝트 생성
- https://console.firebase.google.com 접속
- 프로젝트 추가: SelectMoney
- Android 앱 추가:
  - 패키지명: com.ezcorp.selectmoney
  - SHA-1: 위에서 복사한 값 입력
- google-services.json 다운로드 → app/ 폴더에 복사

### 4. Firebase 서비스 활성화
- Authentication > 익명(Anonymous) 활성화
- Authentication > Google 활성화
- Firestore Database > 데이터베이스 만들기 > 테스트 모드
  - 위치: asia-northeast3 (서울)

### 5. strings.xml 수정
- google-services.json 파일에서 oauth_client 중 type: 3인 client_id 찾기
- app/src/main/res/values/strings.xml 열기
- YOUR_WEB_CLIENT_ID_HERE를 해당 값으로 교체

### 6. 앱 실행 테스트
- 에뮬레이터 또는 실제 기기에서 실행
- 알림 접근 권한 설정 확인
